# Install and configure Hyprland

### Prerequisites
- base-devel
- git
- yay
- kitty

### Install
- [Hyprland Wiki for more info](https://wiki.hyprland.org/)
- Install Hyprland
```
yay -S hyprland
```
- Clone Hyprland config from [here]() to ```~/.config/hypr/```
